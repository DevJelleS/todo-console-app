\# Todo Console App (.NET 10)



A simple console-based To-Do application built with C# and .NET 10.  

This project was created to practice clean code, application structure, and basic persistence.



\## Features

\- Add new to-do items

\- View all to-do items

\- Mark items as completed

\- Delete items

\- Persistent storage using a JSON file



\## Technologies

\- C#

\- .NET 10

\- Console Application

\- System.Text.Json



\## Project Structure

TodoApp/

├── Models/

│ └── TodoItem.cs

├── Services/

│ └── TodoService.cs

├── Program.cs





\## How to Run

1\. Open the solution in Visual Studio 2026

2\. Make sure `.NET 10` is installed

3\. Run the application (`Ctrl + F5`)



The application will automatically create and manage a local `todos.json` file.



\## Learning Goals

\- Working with C# classes and collections

\- Handling user input and validation

\- Applying separation of concerns

\- Using JSON for simple persistence

\- Structuring a small .NET application
